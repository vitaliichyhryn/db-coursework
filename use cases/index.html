              Сценарії використання

              IDCreateAccount
              Назва:Створити обліковий запис (Create Account)
              Учасники:Користувач, система
              Передумови:Користувач не має облікового запису в системі
              Результат:Зареєстрований обліковий запис
              Виключні ситуації:- : Користувач ввів неправильні дані
              - : Користувач намагається створити обліковий запис, який вже існує в системі
              IDLogIntoAccount
              Назва:Увійти в обліковий запис (Log Into Account)
              Учасники:Користувач(працівник команди, керівник команди), система
              Передумови:- Користувач має обліковий запис
              - Користувач повинен бути не авторизованим
              Результат:Успішний вхід в систему
              Виключні ситуації:- : Користувач ввів некоректні дані (невірна електронна пошта або пароль)
              - : Користувач не зареєстрований
              IDEditProfile
              Назва:Редагувати профіль (Edit Profile)
              Учасники:Користувач(працівник команди, керівник команди), система
              Передумови:- Користувач має обліковий запис у системі
              - Користувач авторизований
              Результат:Успішно оновлений профіль
              Виключні ситуації:- : Користувач ввів некоректні дані
              IDDeleteProfile
              Назва:Видалити профіль (Delete Profile)
              Учасники:Користувач(працівник команди, керівник команди), система
              Передумови:Користувач має обліковий запис у системі та авторизований
              Результат:Видалений профіль користувача
              Виключні ситуації:- : Користувач скасував підтвердження видалення
              IDCreateProject
              Назва:Створити проєкт (Create Project)
              Учасники:Керівник команди, система
              Передумови:Керівник команди авторизований
              Результат:Створений новий проєкт
              Виключні ситуації:- : Керівник команди ввів некоректні дані
              IDEditProject
              Назва:Редагувати проєкт (Edit Project)
              Учасники:Керівник команди, система
              Передумови:Керівник команди авторизований
              Результат:Оновлений проєкт
              Виключні ситуації:- : Користувач ввів некоректні дані
              - : Користувач скасував підтвердження змін
              IDDeleteProject
              Назва:Видалити проєкт (Delete Project)
              Учасники:Керівник команди, система
              Передумови:Керівник команди авторизований
              Результат:Проєкт успішно видалено
              Виключні ситуації:- : Користувач скасував підтвердження видалення
              IDAddUserToProject
              Назва:Додати користувача до проєкту (Add User To Project)
              Учасники:Керівник команди, система
              Передумови:- Користувач є адміністратором або власником проєкту
              - Проєкт існує в системі
              - Користувач, якого додають, зареєстрований у системі
              Результат:Новий учасник команди
              Виключні ситуації:- : Неправильно вказані дані користувача
              - : Користувач вже є учасником проєкту
              IDRemoveUserFromProject
              Назва:Видалити користувача з проєкту (Remove User From Project)
              Учасники:Керівник команди, система
              Передумови:- Користувач є адміністратором або власником проєкту
              - Користувач, якого видаляють, є учасником проєкту
              Результат:Видалений учасник з проєкту
              Виключні ситуації:- : Керівник команди скасував підтвердження видалення
              IDCreateTask
              Назва:Створити завдання (Create Task)
              Учасники:Керівник команди, система
              Передумови:- Керівник команди є адміністратором або власником проєкту
              - Керівник команди авторизований
              Результат:Створене завдання
              Виключні ситуації:- : Некоректно введені дані для завдання
              IDEditTask
              Назва:Редагувати завдання (Edit Task)
              Учасники:Користувач(керівник команди, учасник проєкту), система
              Передумови:- Користувач має доступ до проєкту та до завдання
              - Завдання існує в системі
              Результат:Оновлене завдання
              Виключні ситуації:- : Користувач ввів некоректні дані
              - : Користувач скасував підтвердження змін
              IDDeleteTask
              Назва:Видалити завдання (Delete Task)
              Учасники:Користувач(керівник команди, учасник проєкту), система
              Передумови:- Користувач має доступ до проєкту та до завдання
              - Завдання існує в системі
              Результат:Видалене завдання
              Виключні ситуації:- : Користувач скасував підтвердження видалення
              IDCommentTask
              Назва:Додати коментар до завдання (Comment Task)
              Учасники:Користувач(керівник команди, учасник проєкту), система
              Передумови:- Користувач має доступ до проєкту та завдання
              - Завдання існує в системі
              Результат:Новий коментар до завдання
              Виключні ситуації:- : Завдання не знайдено
              IDFilterTasks
              Назва:Відфільтрувати завдання
              Учасники:Користувач (керівник команди, працівник команди), система
              Передумови:- Користувач авторизований
              Результат:Перелік завдань, відібраних за обраними фільтрами
              Виключні ситуації:- Відсутні
              IDBlockProject
              Назва:Заблокувати проєкт
              Учасники:Адміністратор системи, система
              Передумови:- Адміністратор системи авторизований
              - Проєкт активний
              - Проєкт порушує умови використання
              Результат:Заблокований проєкт
              Виключні ситуації:- : Адміністратор скасував операцію блокування
              IDUnblockProject
              Назва:Розблокувати проєкт
              Учасники:Адміністратор системи, система
              Передумови:- Адміністратор системи авторизований
              - Проєкт заблокований
              Результат:Розблокований проєкт
              Виключні ситуації:- : Адміністратор скасував операцію розблокування
              IDBanUser
              Назва:Заблокувати обліковий запис користувача
              Учасники:Адміністратор системи, система
              Передумови:- Адміністратор системи авторизований
              - Обліковий запис користувача активний
              - Користувач порушує умови використання
              Результат:Заблокований обліковий запис користувач
              Виключні ситуації:- : Адміністратор скасував операцію блокування
              IDUnbanUser
              Назва:Розблокувати користувача
              Учасники:Адміністратор системи, система
              Передумови:- Адміністратор системи авторизований
              - Обліковий запис користувача заблокований
              Результат:Розблокований обліковий запис користувача
              Виключні ситуації:- : Адміністратор скасував операцію розблокування
              IDUserSupport
              Назва:Отримати користувацьку підтримку
              Учасники:Користувач (керівник команди, працівник команди), адміністратор системи, система
              Передумови:- Користувач авторизований
              - Користувач хоче звернутися до користувацької підтримки
              Результат:Оброблене звернення користувача
              Виключні ситуації:- : користувач приклав завеликий файл або файл з некоректним розширенням
